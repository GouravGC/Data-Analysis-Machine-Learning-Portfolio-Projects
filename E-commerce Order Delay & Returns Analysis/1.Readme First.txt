## E-commerce Order Delay & Returns Analysis

📊 This project explores return rate by category, return rate by vendor , expense forecast and payment methods usage using Simulation data.

- 📁 Dataset: Simulation dataset
- 🛠 Tools: Pandas, Seaborn, Matplotlib, Plotly
- 📈 Plots: total expenses by category, monthly income expenses net savings , average delivery delay by region , delivery delay days by product category and return rate vs delay orders
